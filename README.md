# Prodesk IT - Digital Agency Landing Page

## 📌 Project Overview
This project was developed as part of a Front-End Developer task submission.

The objective was to build a fully responsive and modern landing page for a fictional digital agency named **Prodesk IT**. The website focuses on clean UI, performance optimization, and user experience.

---

## 🎯 Features
- Responsive Navbar (Desktop + Mobile)
- Hero Section with Call-to-Action
- About Section
- Services Section
- Contact Form
- Dark Mode Toggle
- Glassmorphism UI Effects
- Performance Optimized (LCP improvements)

---

## 🛠️ Technologies Used
- HTML5
- Tailwind CSS
- JavaScript
